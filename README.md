# AI Financial Mentor and Advisor Chatbot

AI-powered financial guidance chatbot developed using Natural Language Processing (NLP) and Flask. The system helps users with financial advice, EMI calculations, ROI calculations, and personalized financial assistance through conversational interaction.

---

# Project Overview

The AI Financial Mentor and Advisor Chatbot is designed to provide users with basic financial guidance in an interactive way. The chatbot analyzes user queries, performs financial calculations, and generates helpful responses using NLP techniques.

The system includes features such as EMI calculation, ROI estimation, financial query handling, and secure user authentication.

---

# Features

- AI-based financial chatbot
- EMI calculation
- ROI and interest calculation
- NLP-based query processing
- Secure JWT authentication
- Responsive web interface
- Real-time user interaction

# Project Workflow

1. User enters financial query
2. NLP processes user input
3. Financial calculations are performed
4. Chatbot generates personalized response
5. Results displayed through web interface
